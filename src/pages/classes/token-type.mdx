## A classe TOKEN_TYPE [#token-type]

<Callout type="error">
    Quaisquer tokens inseridos no código, que não esteja definido nessa classe (na nossa
    linguagem) resultará em um erro disparado pelo analisador léxico
</Callout>

A classe token_type é uma classe que contém todos os tipos de tokens que podemos ter na
nossa linguagem. Quaisquer outro token inserido.

Basicamente é uma tupla que contém um identificador numérico, e uma string com uma representação
visual do nosso token.
